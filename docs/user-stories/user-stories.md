# User Stories

## Sung Jinwoo

User Story 1:
As a frequent business traveller,  
I want to book a flight quickly,  
so I can do it in between meetings without wasting time.

---

User Story 2:
As a frequent business traveller,  
I want to change my flight without cancelling the whole booking,  
so I don’t have to start everything again.

---

User Story 3:
As a frequent business traveller,  
I want to see the full price (including baggage and extra costs) before paying,  
so there are no surprises later.

---

User Story 4:
As a frequent business traveller,  
I want my passenger details saved,  
so I don’t have to re-enter them every time I book.

---

User Story 5:
As a frequent business traveller,  
I want to download an invoice after booking,  
so I can submit it for expenses.

---

User Story 6:
As a frequent business traveller,  
I want to view my previous bookings,  
so I can manage or change them easily.

## Maria

User Story 1:
As a budget-conscious student,
I want to quickly find the cheapest flights,
so I can stay within my limited budget.

---

User Story 2:
As a budget-conscious student,
I want to see the full price (including baggage and extra costs) before paying,
so I don’t get unexpected charges.

---

User Story 3:
As a budget-conscious student,
I want simple filters (price, time, duration),
so I can easily narrow down my options.

---

User Story 4:
As a budget-conscious student,
I want a fast and simple booking process on my phone,
so I can book without getting overwhelmed.

## David 

User Story 1:
As an older traveller with poor eyesight,
I want the text and buttons to be large and clear,
so I can easily read and navigate the website without strain.

---

User Story 2:
As an older traveller who is not confident with technology,
I want a simple and guided booking process,
so I don’t get confused or make mistakes while booking.

---

User Story 3:
As an older traveller planning a holiday,
I want to clearly compare flight options (comfort vs price),
so I can choose what suits me and my wife best.

---

User Story 4:
As an older traveller with limited technical skills,
I want clear instructions and confirmations at each step,
so I feel confident that I am booking correctly.

## Sophie 

User Story 1:
As an airline employee managing bookings,
I want to view and track all reservations in one place,
so I can quickly understand the current booking status.

---

User Story 2:
As an airline employee,
I want to easily modify bookings (dates, passenger details, seats),
so I can handle customer requests efficiently.

---

User Story 3:
As an airline employee,
I want to quickly process cancellations and availability changes,
so the system stays accurate and up to date.

---

User Story 4:
As an airline employee who uses the system for long hours,
I want a dark mode or low-brightness interface,
so I can reduce eye strain while working.

===

User Story 5:
As an airline employee,
I want clear confirmation prompts before making changes,
so I don’t accidentally make errors in bookings.
