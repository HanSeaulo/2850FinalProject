# User Stories

## Sung Jinwoo

User Story 1:
As a frequent business traveller,  
I want to book a flight quickly,  
so I can do it in between meetings without wasting time.

---

User Story 2:
As a frequent business traveller,  
I want to change my flight without cancelling the whole booking,  
so I don’t have to start everything again.

---

User Story 3:
As a frequent business traveller,  
I want to see the full price (including baggage and extra costs) before paying,  
so there are no surprises later.

---

User Story 4:
As a frequent business traveller,  
I want my passenger details saved,  
so I don’t have to re-enter them every time I book.

---

User Story 5:
As a frequent business traveller,  
I want to download an invoice after booking,  
so I can submit it for expenses.

---

User Story 6:
As a frequent business traveller,  
I want to view my previous bookings,  
so I can manage or change them easily.
